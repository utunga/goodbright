Kōrero Māori

  • Automated Speech Recognition
  • Deep learning / machine learning

Working with iwi group TeHiku.NZ I led the machine learning effort to train models and build an API for the world's most accurate automated speech recognition (ASR) system for Te Reo Māori. When this system was presented at language conferences in Oxford and Cambridge in 2018, senior developers from Mozilla described themselves as "shocked and amazed" at the high accuracy of our model. Te Hiku's app using their model is available at KōreroMāori.com

I am delighted to report that my work with Te Hiku continues into the new year.

Deep PBR

  • Machine Learning/AI
  • VR/AI

Working with the VR pioneers at realityVirtual I designed, trained and deployed a new machine learning pipeline and API for creating virtual reality assets (displacement maps, normals and the like) from regular SLR photographs. As of early 2019 this project has recently secured significant interest from multiple investors.
